About the Game

This mini game is a Unity-made teleportation platformer built around timing and quick decisions. You jump across randomly generated floating platforms using a fast teleport ability. The game includes two modes: Infinite Mode, where you try to survive as long as possible, and Timed Mode, where you score as much as you can in 60 seconds. It uses procedural spawning, simple player physics, C# scripting, UI management, and basic data saving for high scores. This webpage includes screenshots, gameplay details, a video demo, and a downloadable build. The project highlights core game-development skills and a focus on responsiveness and replayability.
